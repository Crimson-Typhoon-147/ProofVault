# ProofVault

A blockchain-based Evidence Integrity, Verification, and Records Ledger system.
Built using:

- React (Frontend)
- TailwindCSS
- IPFS for decentralized storage
- SHA-256 hashing
- Modern enterprise-style UI (Axon + Kaleido inspired)

## Features (In Progress)
- Upload and hash files
- IPFS storage
- Verification engine
- Immutable records ledger
- Enterprise dashboard and UI

## Project Structure
ProofVault/
 └── frontend/   → React + TailwindCSS UI

Backend (blockchain + IPFS integration) will be added soon.
